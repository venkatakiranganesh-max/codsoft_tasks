# Smart Rule Based Chatbot

## Project Description

Smart Rule Based Chatbot is a Python-based chatbot application that provides responses to user queries using predefined rules and pattern matching techniques.

This project helps to understand the basics of Natural Language Processing (NLP) and chatbot conversation flow.

## Features

- Interactive chatbot conversation
- Rule-based response generation
- Pattern matching technique
- Random response selection
- User-friendly messages
- Exit command option

## Technologies Used

- Python
- Pattern Matching
- Rule-Based Logic
