# Image Captioning AI

## Description
This project combines Computer Vision and Natural Language Processing to generate captions for images automatically.

The system uses a pre-trained BLIP transformer model to understand image features and create meaningful text descriptions.

## Technologies Used
- Python
- PyTorch
- Transformers
- Hugging Face BLIP Model
- Pillow

## Project Structure

Image_Captioning_AI
│
├── app.py
├── caption_generator.py
├── images/
├── output/
├── requirements.txt
└── README.md

## How to Run

Install requirements:

pip install -r requirements.txt

Run:

python app.py

Enter image path:

images/test.jpg
