# Tic-Tac-Toe AI

## Project Description

This project implements an Artificial Intelligence (AI) agent that plays the classic Tic-Tac-Toe game against a human player.

The AI player uses the Minimax Algorithm with Alpha-Beta Pruning optimization to analyze possible moves and select the best move. This makes the AI player intelligent and difficult to defeat.

This project demonstrates basic concepts of Artificial Intelligence, Game Theory, and Search Algorithms.

## Algorithms Used

### Minimax Algorithm

Minimax is a decision-making algorithm used in game theory. It explores all possible moves and selects the move that gives the maximum chance of winning for the AI player.

### Alpha-Beta Pruning

Alpha-Beta Pruning improves the Minimax algorithm by removing unnecessary branches from the search process. It reduces computation time and makes the AI faster.
